# Online-Reservation-System

A professional and colorful **Online Reservation System** developed using **HTML, CSS, and JavaScript**. This project allows users to log in, reserve train tickets, generate a unique PNR number, search reservations, and cancel booked tickets.

---

## 📌 Project Overview

The Online Reservation System is a simple web application designed for learning front-end web development. It demonstrates the use of HTML for structure, CSS for styling, and JavaScript for client-side functionality.

---

## ✨ Features

- 🔐 User Login
- 🚆 Train Ticket Reservation
- 🎫 Automatic PNR Number Generation
- 💾 Data Storage using Local Storage
- 🔍 Search Reservation by PNR
- ❌ Cancel Ticket
- 📱 Responsive Design
- 🎨 Modern Colorful User Interface
- ✔ Form Validation

## 🚆 Reservation Process

1. Login using the demo credentials.
2. Enter passenger details.
3. Enter train information.
4. Select journey date and class.
5. Click **Book Ticket**.
6. A unique **PNR Number** is generated.
7. Reservation details are saved in Local Storage.

---

## ❌ Ticket Cancellation

1. Open the Cancellation page.
2. Enter the generated PNR Number.
3. Click **Search**.
4. Reservation details will be displayed.
5. Click **Cancel Ticket** to delete the booking.

---

## 📱 Form Validation

The application validates:

- Required fields
- Mobile number (10 digits)
- Journey date
- From and To stations should be different

---

## 💾 Data Storage

This project uses the browser's **Local Storage** to store reservation details.

No database or server is required.
